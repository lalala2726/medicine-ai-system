# 鱼骨图（Fishbone Diagram）

也称石川图，用于识别和展示问题的根本原因，形似鱼骨。根节点为问题描述，子节点为各原因分类及具体原因。

## 字段说明

| 字段              | 必填 | 类型     | 说明                                   |
|-----------------|:--:|--------|--------------------------------------|
| `data`          |    | 对象     | 图表数据，递归树形结构                          |
| `data.name`     |    | 文本     | 问题/原因分类名称（根节点为问题，子节点为原因分类）           |
| `data.children` |    | 数组对象   | 原因列表（递归结构）                           |
| `title`         |    | 文本     | 图表标题                                 |
| `theme`         |    | 文本     | 主题，可选 `default` / `dark` / `academy` |
| `height`        |    | 数值（像素） | 图表高度                                 |

## 输出格式

````
```fishbonediagram
{JSON配置}
```
````

## 示例

```fishbonediagram
{
  "theme": "default",
  "height": 600,
  "data": {
    "name": "订单延迟",
    "children": [
      {
        "name": "人",
        "children": [
          { "name": "新员工培训不足" },
          { "name": "排班不均" }
        ]
      },
      {
        "name": "流程",
        "children": [
          { "name": "审核环节过多" },
          { "name": "异常单处理慢" }
        ]
      },
      {
        "name": "系统",
        "children": [
          { "name": "库存同步延迟" },
          { "name": "告警不及时" }
        ]
      }
    ]
  }
}
```
