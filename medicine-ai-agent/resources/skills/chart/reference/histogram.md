# 直方图（Histogram）

显示连续型数据的分布情况，柱子间无空隙，X 轴表示数值区间。注意：`data` 为原始数值数组（非区间统计结果），系统会自动分箱。

## 字段说明

| 字段           | 必填 | 类型     | 说明                                   |
|--------------|:--:|--------|--------------------------------------|
| `data`       |    | 数值数组   | 原始数据（非区间统计结果）                        |
| `binNumber`  |    | 数值     | 区间个数                                 |
| `axisXTitle` |    | 文本     | X轴标题                                 |
| `axisYTitle` |    | 文本     | Y轴标题（通常为"频数"或"频率"）                   |
| `title`      |    | 文本     | 图表标题                                 |
| `theme`      |    | 文本     | 主题，可选 `default` / `dark` / `academy` |
| `height`     |    | 数值（像素） | 图表高度                                 |

## 输出格式

````
```histogram
{JSON配置}
```
````

## 示例

```histogram
{
  "title": "订单金额分布",
  "axisXTitle": "金额区间",
  "axisYTitle": "频数",
  "binNumber": 8,
  "height": 400,
  "data": [12, 15, 18, 20, 21, 25, 30, 35, 40, 42, 45, 50, 58, 66, 72]
}
```
