# 漏斗图（Funnel）

展示多阶段数据的转化或流失情况，顶部宽底部窄。适合电商转化漏斗、用户行为路径等场景。

## 字段说明

| 字段                | 必填 | 类型     | 说明                                   |
|-------------------|:--:|--------|--------------------------------------|
| `data`            |    | 数组对象   | 图表数据，按阶段顺序排列                         |
| `data[].category` |    | 文本     | 阶段名称                                 |
| `data[].value`    |    | 数值     | 数值                                   |
| `title`           |    | 文本     | 图表标题                                 |
| `theme`           |    | 文本     | 主题，可选 `default` / `dark` / `academy` |
| `height`          |    | 数值（像素） | 图表高度                                 |

## 输出格式

````
```funnel
{JSON配置}
```
````

## 示例

```funnel
{
  "theme": "default",
  "height": 300,
  "data": [
    { "category": "浏览", "value": 1000 },
    { "category": "加购", "value": 620 },
    { "category": "下单", "value": 340 },
    { "category": "支付", "value": 260 }
  ]
}
```
