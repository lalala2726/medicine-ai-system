# 词云图（Word Cloud）

展示文本数据中词语的频率或权重，字体大小表示重要性/出现频次。

## 字段说明

| 字段             | 必填 | 类型     | 说明                                   |
|----------------|:--:|--------|--------------------------------------|
| `data`         |    | 数组对象   | 图表数据                                 |
| `data[].text`  |    | 文本     | 词汇                                   |
| `data[].value` |    | 数值     | 重要性分数（频次/权重）                         |
| `title`        |    | 文本     | 图表标题                                 |
| `theme`        |    | 文本     | 主题，可选 `default` / `dark` / `academy` |
| `height`       |    | 数值（像素） | 图表高度                                 |

## 输出格式

````
```wordcloud
{JSON配置}
```
````

## 示例

```wordcloud
{
  "theme": "default",
  "height": 400,
  "data": [
    { "text": "发货快", "value": 88 },
    { "text": "包装好", "value": 73 },
    { "text": "价格实惠", "value": 64 },
    { "text": "客服专业", "value": 52 }
  ]
}
```
