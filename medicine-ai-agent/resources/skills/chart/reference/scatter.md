# 散点图（Scatter）

显示两个数值变量之间的关系，每个点代表一个数据项。支持通过 `group` 字段进行多组数据对比。

## 字段说明

| 字段             | 必填 | 类型     | 说明                                   |
|----------------|:--:|--------|--------------------------------------|
| `data`         |    | 数组对象   | 图表数据                                 |
| `data[].x`     |    | 数值     | X轴数值                                 |
| `data[].y`     |    | 数值     | Y轴数值                                 |
| `data[].group` |    | 文本     | 分组名称，用于多组数据对比                        |
| `axisXTitle`   |    | 文本     | X轴标题                                 |
| `axisYTitle`   |    | 文本     | Y轴标题                                 |
| `title`        |    | 文本     | 图表标题                                 |
| `theme`        |    | 文本     | 主题，可选 `default` / `dark` / `academy` |
| `height`       |    | 数值（像素） | 图表高度                                 |

## 输出格式

````
```scatter
{JSON配置}
```
````

## 示例

```scatter
{
  "axisXTitle": "访客数",
  "axisYTitle": "下单数",
  "height": 400,
  "data": [
    { "x": 120, "y": 18 },
    { "x": 180, "y": 29 },
    { "x": 220, "y": 34 },
    { "x": 260, "y": 41 }
  ]
}
```
