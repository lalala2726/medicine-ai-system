# 饼图（Pie）

展示各部分占整体的比例，扇形大小表示占比。设置 `innerRadius` 可变为环图。

## 字段说明

> **所有字段均为顶层字段，禁止使用 `config`、`options` 等嵌套包装对象。**

| 字段                | 必填 | 类型     | 说明                                   |
|-------------------|:--:|--------|--------------------------------------|
| `data`            |    | 数组对象   | 图表数据                                 |
| `data[].category` |    | 文本     | 分类名称                                 |
| `data[].value`    |    | 数值     | 数值（不可使用百分比数字）                        |
| `innerRadius`     |    | 数值     | 内半径，设置后变为环图（如 0.6）                   |
| `title`           |    | 文本     | 图表标题（**顶层字段，不可嵌套在 config 内**）        |
| `theme`           |    | 文本     | 主题，可选 `default` / `dark` / `academy` |
| `height`          |    | 数值（像素） | 图表高度                                 |

## 输出格式

````
```pie
{JSON配置}
```
````

## 示例

```pie
{
  "theme": "default",
  "height": 400,
  "data": [
    { "category": "已支付", "value": 180 },
    { "category": "待支付", "value": 45 },
    { "category": "已取消", "value": 20 }
  ]
}
```
