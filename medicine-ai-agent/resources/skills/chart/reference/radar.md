# 雷达图（Radar）

展示多维度数据，从中心点等角度射出多个轴，适合综合评价和多指标对比。支持多组数据对比（通过 `group` 字段）。

## 字段说明

| 字段             | 必填 | 类型     | 说明                                   |
|----------------|:--:|--------|--------------------------------------|
| `data`         |    | 数组对象   | 图表数据                                 |
| `data[].name`  |    | 文本     | 维度名称                                 |
| `data[].value` |    | 数值     | 数值                                   |
| `data[].group` |    | 文本     | 分组名称，用于多组数据对比                        |
| `title`        |    | 文本     | 图表标题                                 |
| `theme`        |    | 文本     | 主题，可选 `default` / `dark` / `academy` |
| `lineWidth`    |    | 数值     | 线条宽度                                 |
| `height`       |    | 数值（像素） | 图表高度                                 |

## 输出格式

````
```radar
{JSON配置}
```
````

## 示例

```radar
{
  "theme": "default",
  "lineWidth": 2,
  "height": 400,
  "data": [
    { "name": "履约速度", "value": 82 },
    { "name": "售后满意度", "value": 76 },
    { "name": "库存健康度", "value": 68 },
    { "name": "毛利率", "value": 73 },
    { "name": "复购率", "value": 79 }
  ]
}
```
